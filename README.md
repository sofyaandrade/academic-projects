# Academic Projects

Portfólio de projetos desenvolvidos durante minha graduação em Sistemas de Informação.

Este repositório reúne aplicações acadêmicas criadas para explorar desenvolvimento web e mobile, bancos de dados, comunicação em tempo real, programação lógica, padrões de projeto e integração com serviços em nuvem.

## Projetos

### Desenvolvimento Mobile

#### [InsuGuia Clinical Prototype](./mobile-development/insuguia-clinical-prototype)

Protótipo acadêmico multiplataforma para acompanhamento de pacientes hospitalizados não críticos, registro de glicemia e estudo de fluxos relacionados à insulinoterapia.

`Flutter` `Dart` `Firebase` `Cloud Firestore` `React` `TypeScript`

Projeto semestral em grupo, desenvolvido a partir de uma demanda acadêmica ligada à pesquisa de doutorado de um professor da área de Medicina. A versão disponível é exclusivamente educacional e não possui validação clínica.

---

#### [Responsive Book Catalog PWA](./mobile-development/responsive-book-catalog-pwa)

Catálogo responsivo de livros que pode ser instalado como aplicativo e acessado offline após o primeiro carregamento.

`HTML5` `CSS Grid` `JavaScript` `PWA` `Service Worker`

---

#### [Travel Agency PWA](./mobile-development/travel-agency-pwa)

Landing page responsiva para uma agência de viagens fictícia, com navegação por seções, layout em Flexbox e suporte a instalação e funcionamento offline.

`HTML5` `CSS Flexbox` `JavaScript` `PWA` `Service Worker`

### Desenvolvimento Web

#### [Fitness Assessment System](./web-development/fitness-assessment-system)

Sistema autenticado para gerenciamento de alunos e avaliações físicas de uma academia, com operações CRUD, busca, paginação e relacionamento entre registros.

`PHP` `Laravel` `Blade` `Bootstrap` `PostgreSQL` `Vite`

---

#### [Hospital Feedback System](./web-development/hospital-feedback-system)

Sistema de pesquisa de satisfação hospitalar com avaliações anônimas de 0 a 10, comentários opcionais e área administrativa para gerenciamento das perguntas.

`PHP` `PostgreSQL` `PDO` `JavaScript` `Fetch API`

### Paradigmas de Programação

#### [Hotel Booking Design Patterns](./programming-paradigm/hotel-booking-design-patterns)

Aplicação de linha de comando para reservas de hotel criada para demonstrar padrões de projeto em um domínio único e testável.

`Python` `pytest` `Factory Method` `Strategy` `Decorator` `Observer` `Singleton`

---

#### [Tax Refund Expert System](./programming-paradigm/tax-refund-expert-system)

Sistema especialista que utiliza fatos e regras lógicas para avaliar solicitações de restituição tributária, determinar prioridade, prazo e departamento responsável.

`Prolog` `SWI-Prolog` `Programação Lógica` `Motor de Inferência`

---

#### [Real-Time Court Booking System](./programming-paradigm/real-time-court-booking-system)

Sistema de reserva de quadras com servidor concorrente em Go e atualização instantânea entre múltiplos clientes por WebSocket.

`Go` `React` `WebSocket` `Goroutines` `Mutex` `Vite`
